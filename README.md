### About

Full-stack engineer working in applied AI. Over the last few years I've moved from frontend
into AI product work, and I like owning a feature across the whole stack rather than handing it
between layers.

At **Tourlane** I designed and own a transport-recommendation service end to end — a
Snowflake/Dagster pipeline → LLM → FastAPI → React — and build LLM services around trip
planning. Before that, ~4 years of full-stack product work (React, Next.js, Ruby on Rails) and,
earlier, leading a frontend team on a large banking app.

**How I work:** eval-driven development for AI behavior, TDD/BDD for code — nothing ships that
didn't have a failing check first. Latency, cost and reliability matter as much as accuracy.

### Featured

**[⚙️ agent-harness](https://github.com/RubLo/agent-harness)** — a reliability harness for AI
coding agents. My way of working made reusable: eval-driven + TDD/BDD discipline enforced
through rules the agent reads first, a memory that survives sessions, and guardrail hooks that
fire deterministically.
